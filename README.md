# BookStore Online(Cửa hàng bán sách trực tuyến)

IDE: Spring Tool Suite 4  
Database: MySQL  
Framework: Spring Boot (dự kiến)  
Frontend: HTML, CSS, JS, Angular (dự kiến), yêu cầu có boostrap 4  
Programming Language: Java  

Chức năng dự kiến:  

1.Tài khoản:  
  * Đăng nhập, đăng xuất.  
  * Quên mật khẩu.  
  * Remember Me.  
  
2.Giỏ hàng:  
  * Thêm, sửa, xóa sản phẩm trong giỏ hàng.  
  
3.Đơn hàng:  
  * Khách hàng áp mã khuyến mãi vào sản phẩm để giảm giá (mã km có thời gian và số lần áp dụng).
  * Khách hàng áp mã miễn phí vận chuyển vào đơn đặt hàng (1 hoặc nhiều sản phẩm).  
  * Khách hàng chỉnh sửa thông tin đơn hàng (tên, địa chỉ, số điện thoại).  
  
4.Thanh toán online:
  * Thanh toán qua API của MoMo.  
  * Thanh toán qua PayPal.  
  * Thanh toán qua ZaloPay.  
  * Thanh toán qua VNPay.  
  
5.Hiển thị sản phẩm:
  * Phân trang.  
  * Tìm kiếm sản phẩm (theo tên, thể loại, địa chỉ shop, giá).  

6.Sản phẩm:  
  * Sản phẩm đã xem.  
  * Gợi ý sản phẩm.  
  * Số lượng người đã mua sản phẩm đó.  
  * Khách có thể đánh giá sao, bình luận sản phẩm, reaction sản phẩm đó.  
  * Chia sẻ sản phẩm lên FB.  
  
7.Còn nữa...
